What We Build

The SIM Lab creates practical tools, research instruments, and learning resources — built to support students and academics while extending our research.

CAGE
Equipment

The Cage

A Next.js production equipment reservation and checkout platform for SIUE Mass Communications students. QR-based checkout workflows integrate with Snipe-IT asset management for full inventory control.

V→V
OER Textbook

Vibes to Variables

An open educational resource guiding undergrads through a complete research project — from curiosity to a reproducible study — using a real dataset of 1,792 songs. Built with Quarto.

View on GitHub
cpkR
R Package

coursepackR

The companion R package for MC-451 Research Methods. Bundles the Vibes to Variables textbook chapters, weekly assignment templates, and a curated music dataset for hands-on methods instruction.

View on GitHub
MSYNC
Qualitative

MethodoSync

A browser-based qualitative research tool for video annotation and codebook generation. Paste transcripts or YouTube links, apply grounded codes, and export structured Markdown codebooks — no server required.

Open WebUI Launcher

A cross-platform desktop launcher (OWL) for self-hosted Open WebUI instances — simplifying local AI deployment for researchers and educators without technical setup overhead.
